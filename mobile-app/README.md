# Expo Mobile App

## Run
1. Install deps: `npm install`
2. Make sure API server runs at `http://192.168.1.7:3000`
3. Start Expo: `npm run start`

## Features
- List mahasiswa with `FlatList`
- Search by NIM or name
- Create mahasiswa
- Update mahasiswa
- Delete mahasiswa with backend rule enforcement
- Refresh after create/update/delete without restart
- Client-side form validation
- Error alert from backend response
